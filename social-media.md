# Social Media Skill — FXEDGE

> A skill file for AI agents (Claude, Codex) to write social media content for FXEDGE.
> Place in repo root or pass as context to AI tools.

---

## Brand Voice

### Who FXEDGE is
- A trading journal built BY a forex trader FOR ICT/SMC traders
- Solo founder, non-developer, built with AI in 1 month
- Authentic, direct, no fluff
- Underdog energy — going up against $30/mo tools

### Tone
- Confident but not arrogant
- Specific not vague
- Show numbers, screenshots, real data
- Build in public — share the journey
- No corporate speak, no hype emojis (except 🔥 sparingly)

### Voice rules
- Short sentences. Punchy.
- Lead with specifics, never generalities
- "I built this" not "We built this"
- Numbers > adjectives ("68% win rate" not "high win rate")
- Trading terms allowed (kill zones, OB, FVG, manipulation)

## Viral Hook Formulas

### Numbers + outcome + timeframe
```
I built [X] in [Y time] with [Z resources].
```
Example: "I built a SaaS app in 1 month with zero coding experience."

### Pain → action
```
I was tired of [pain]. So I [action].
```
Example: "I was tired of paying $30/mo for tools that don't understand ICT. So I built my own."

### Counter-intuitive truth
```
Most people think X. The truth is Y.
```
Example: "Most traders think more trades = more money. My journal proves the opposite."

### Specific number + question
```
[Specific stat]. Here's why →
```
Example: "I'm 68% win rate on Mondays, 23% on Wednesdays. Here's what I figured out:"

### Public reveal
```
I just shipped [X]. Here's what it does:
```

## Thread Structure (proven)

### 7-tweet template
1. **Hook** — image + curiosity gap
2. **Pain** — what problem you had
3. **Honesty** — what you didn't have/know
4. **Process** — how you did it
5. **Result** — what you built (with screenshot)
6. **Lesson** — universal takeaway
7. **CTA** — drop emoji for early access / repost

### Single tweet template
```
[Specific stat or moment]

[1-2 short lines elaborating]

[Optional: image]
```

## Engagement Rules

### Reply within first hour
The algorithm rewards posts with comments early. Reply to every comment in the first 60 minutes after posting — even just "thanks" or 🔥.

### Quote tweet your own content
After 24 hours, quote tweet with a specific stat or angle to revive the post.

### Tag carefully
- Don't tag big accounts in the original tweet (looks needy)
- Tag them in a follow-up reply if relevant

### Pin the best ones
Keep your best thread pinned to your profile.

## What NOT to do

### Avoid
- Generic motivational content
- Excessive emojis
- Selling without showing
- Long preachy posts
- Engagement bait ("RT if you agree")
- Buying followers
- Reposting other people's content without credit

### Don't promise
- "I'll teach you to be profitable"
- "You can make $X/month"
- "This is the only journal you need"
